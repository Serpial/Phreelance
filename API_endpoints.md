## API Endpoints

#### Auctions

* Get all Auctions
* Get Auctions for user ID (watched, created by)
* Get Auction by ID
* Update Auction by ID
* Delete Auction by ID
* Create Auction

#### Bids

* Get bids for Contract ID
* Create bid
* Update bid by id

#### Users

* Create User
* Delete User by ID
* Update User by ID

#### Settings

* Update profile settings by user ID
